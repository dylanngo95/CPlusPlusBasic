# Install project

Make sure the your local workstation was installed: CMake, Make

## Go to project

```bash
    cd <Project Name>
```


## Display the tool may be using build c ++

```bash
    cmake --help
```

## Run CMake

```bash
    cmake -G "MinGW Makefiles"
```

## Build project

```bash
    make
```

## Run program in window

```bash
    ./program_name.exe
```
